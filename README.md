# Bucket List Cinema (影迷願望清單) - Web App MVP 🎬

這是一個結合了 **「個人化觀影心得日誌」** 與 **「社群每週策展主題投票機制」** 的全端 Web 應用程式原型 (MVP)。
本專案為自動建置腳本生成。

## 🚀 快速啟動指南

1. 建立虛擬環境與安裝依賴 (終端機指令)
    python -m venv venv
    # Windows: venv\Scripts\activate
    # macOS/Linux: source venv/bin/activate
    pip install -r requirements.txt

2. 啟動 Web 伺服器
    uvicorn main:app --reload

3. 打開瀏覽器訪問 [http://127.0.0.1:8000](http://127.0.0.1:8000) 即可體驗！
